# Category Counter Service

Микросервис для парсинга количества объявлений в категориях Cian и записи результатов в Google Sheets.

## Функционал

Парсит 4 категории:
- Вторичка Москва
- Первичка Москва
- Первичка МО
- Вторичка МО

Для каждой категории извлекает количество объявлений и записывает результаты в Google Sheets (таб "Balans").

## Запуск

```bash
# Через docker-compose
docker-compose --profile manual up category_counter

# Или напрямую
docker-compose run --rm category_counter
```

## Переменные окружения

Необходимые переменные в `.env`:

```env
# Google Sheets
SPREADSHEET_ID=your_spreadsheet_id

# Прокси (опционально, через запятую)
PROXIES=http://proxy1:port,http://proxy2:port
```

## Как работает

1. Берет все 4 ссылки на категории
2. Для каждой категории:
   - Делает запрос через curl_cffi с мобильным прокси (если настроен)
   - Получает HTML
   - Конвертирует HTML в Markdown через go-html-to-md сервис
   - Ищет паттерн "Найдено {n} объявлений"
   - Извлекает количество
3. Записывает все результаты в Google Sheets в новую колонку с датой

## Структура записи в Google Sheets

```
Дата: 19.03.2026 14:30:00
Вторичка Москва: 123456
Первичка Москва: 234567
Первичка МО: 345678
Вторичка МО: 456789
Всего: =SUM(B2:E2)
Точка равновесия: 150000
```

Данные записываются в строки (не в столбцы):
- Колонка A: Дата и время
- Колонка B: Вторичка Москва
- Колонка C: Первичка Москва
- Колонка D: Первичка МО
- Колонка E: Вторичка МО
- Колонка F: Всего (формула суммы)
- Колонка G: Точка равновесия (150000)

## Зависимости

- html_to_markdown сервис (для конвертации HTML в Markdown)
- Google Sheets API (credentials.json)
- curl_cffi (для запросов с имитацией браузера)
